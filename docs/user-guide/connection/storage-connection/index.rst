Storage Connections
==============

Sparkflows enables you to connect to multiple databases and storage systems by creating the Storage Connection to them.

In Sparkflows, **Storage Connection** can connect to the following data sources:

- Box
- Cassandra
- ClickHouse
- Confluence
- JDBC
- MongoDB
- MySQL
- Oracle
- Oracle Netsuite
- PostgreSQL
- Redshift
- Salesforce
- SAP HANA
- ServiceNow
- SFTP
- SharePoint
- Shopify
- Snowflake
- SQL Server
- Sybase
- TeraData
- Trino

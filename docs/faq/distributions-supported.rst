Distributions Supported
-----------------------

What distributions or platforms are supported with Sparkflows?
===========

Sparkflows has been tested with GCP Dataproc, Databricks, AWS EMR, AWS Glue, Cloudera, Kubernetes, Apache Spark distributions.
 
*Note: Any cluster with Apache Spark 2.3+ is compatible with Sparkflows.*

Can I run Sparkflows on my Amazon AWS cluster or Microsoft Azure or Google Cloud?
===========

Yes, Sparkflows only requires an Apache Spark cluster for successful deployment, making it compatible with Amazon AWS, Microsoft Azure, and Google Cloud.

Copilot Additional Features 
=========================================

This document describes the additional features available in Copilot within Sparkflows in both the Workflow Designer and the Pipeline Designer, including Feedback, Reports, and Report Email Configuration.

     :width: 60%

Copilot Report
------------

 .. figure:: ../../_assets/user-guide/copilot/Copilot-Add-On-Features/copilot-report-feedback.png
     :alt: copilot features
     :width: 60%

**Add Report**
+++++++++++++++++++

* When a user clicks on **Report**, they can report an issue with the Copilot response.

 .. figure:: ../../_assets/user-guide/copilot/Copilot-Add-On-Features/copilot-add-report.png
     :alt: copilot features
     :width: 60%

* This feature is used to flag incorrect and problematic responses generated by Copilot.



**View Report**
+++++++++++++++++++

* Admins can view user reports in the Copilot window under: **Administration --> Reports** tab

 .. figure:: ../../_assets/user-guide/copilot/Copilot-Add-On-Features/copilot-report-list.png
     :alt: copilot features
     :width: 60%

* This section lists all reported issues for further investigation and action.


Configuring Report Emails
------------

* Users can configure email addresses while creating or updating a Copilot. When an issue is reported, the report will be sent to the configured email addresses.

 .. figure:: ../../_assets/user-guide/copilot/Copilot-Add-On-Features/copilot-report-email-config.png
     :alt: copilot features
     :width: 60%

* Once a user submits a report, an email notification will be sent to the configured recipients, as shown below.

Health Check
============

The following URL can be used for checking the health of Sparkflows

::
   
    http://localhost:8080/actuator/health 

once the above URL is added in Browser the response should be something like below ::

    {"status":"UP"}

.. figure:: ..//_assets/operating/healthcheck.PNG
   :alt: operating
   :width: 60%
    
.. note::  Make sure to change localhost to your domain name or your ip where Sparkflows is running

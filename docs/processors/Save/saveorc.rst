SaveORC
=========== 

Saves the DataFrame into the specified location in ORC Format

Type
--------- 

transform

Class
--------- 

fire.nodes.save.NodeSaveORC

Fields
--------- 

.. list-table::
      :widths: 10 5 10
      :header-rows: 1
      * - Name
        - Title
        - Description
      * - path
        - Path
        - Path where to save the ORC files
      * - saveMode
        - Save Mode
        - Whether to Append, Overwrite or Error if the path Exists

Storage Connections
==============

Sparkflows enable you to connect to multiple databases and storage systems by creating the Storage Connection to them.

In Sparkflows ``Storage Connection`` can connect to the following data sources:

* Cassandra
* Hive
* MongoDB
* MySQL
* PostgreSQL
* Redshift
* SQL Server
* Snowflake
